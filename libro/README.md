# Estructura del libro

Este directorio recoge el manuscrito de *Aventuras de conocimiento*. La organización busca separar el prólogo, las ideas principales y las futuras secciones del libro para facilitar su desarrollo incremental.

## Secciones

- `prologo/`: contiene el texto completo del prólogo, el listado de ideas centrales, una frase que resume el espíritu del libro y la caracterización del tipo de obra.
- `capitulos/` (por crear): espacio reservado para los capítulos principales. Cada capítulo podrá alojarse en un subdirectorio propio con su borrador y notas de investigación.
- `anexos/` (por crear): espacio para material de apoyo como glosarios, referencias y guías prácticas.

## Próximos pasos sugeridos

1. Definir la tabla de contenidos inicial en `capitulos/` con títulos y objetivos por capítulo.
2. Añadir notas de trabajo y referencias cruzadas en `anexos/` para mantener el contexto de investigación.
3. Establecer una convención para versiones y revisiones de cada texto (por ejemplo, numerar iteraciones o usar comentarios en línea).
